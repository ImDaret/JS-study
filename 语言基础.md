# 变量

## var 关键字

### var 声明的作用域

使用 var 定义的变量将会变成该函数内部的局部变量，比如下面定义的变量将在函数退出时被销毁：

```javascript
function test() {
  var message = "hi";
}
test();
console.log(message); //出错！;
```

假如不用 var 关键字声明，该变量将会变成全局变量

```javascript
function test() {
  message = "hi";
}
test();
console.log(message); // “hi”;
```
